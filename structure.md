# Описание проекта

**pw_launcher/** — лаунчер для игры, предоставляющий удобный графический интерфейс для управления профилями, персонажами, настройками запуска и мониторингом дебаффов.

## Структура проекта

pw_launcher/
│
├── main.py                  # Точка входа, запуск приложения
├── styles.py                # Оформление, цвета, шрифты, тема интерфейса
├── utils.py                 # Общие вспомогательные функции (работа с профилями, настройками и пр.)
│
├── ui/                      # Модули интерфейса
│   ├── main_menu.py         # Главное меню программы
│   ├── character_menu.py    # Менеджер персонажей
│   ├── add_character_menu.py# Добавление нового персонажа
│   ├── profile_menu.py      # Менеджер игровых профилей
│   ├── settings_menu.py     # Окно настроек (пути, интеграция и др.)
│   └── debuff_monitor.py    # Окно мониторинга дебаффов (реалтайм-стрим из игры)
│
└── assets/                  # Вспомогательные файлы и ресурсы
    └── icon.ico            # Значок приложения
